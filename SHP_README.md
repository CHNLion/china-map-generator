# 中国行政区划数据文件说明

这个目录应包含以下SHP格式的地图数据文件：

- `省.shp`：省级行政区划边界数据
- `市.shp`：市级行政区划边界数据
- `县.shp`：县级行政区划边界数据

以及相应的辅助文件（.dbf, .shx, .prj, .sbn, .sbx等）。

## 文件放置说明

1. 下载全部地图数据文件
2. 解压缩后，将所有文件放置在项目的`shp/`目录下
3. 确保文件名称正确，应为"省.shp"、"市.shp"和"县.shp"等

## 数据说明

这些数据包含中国行政区域的空间地理信息，支持以下字段：

### 省级数据
- 省：省份名称
- 省代码：省份编码
- 类型：行政区划类型

### 市级数据
- 市：城市名称
- 省：所属省份

### 县级数据
- NAME：县区名称
- 市：所属城市
